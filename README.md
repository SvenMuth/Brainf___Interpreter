# Brainf___Interpreter in C

An interpreter for Brainf*ck. Only tested on Linux.

## Usage
    ./BrainfuckInterpreter [RUNNING_MODE] filename
                            -DEBUG     Run debug mode
                            -NO_LOG    Only print output without execution logging

## Building

    $ mkdir build
    $ cd build
    $ cmake ..
    $ make
